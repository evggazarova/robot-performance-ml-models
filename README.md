###  Описание проекта
Практическая работа по применению методов машинного обучения для анализа производительности человекоподобных роботов.

### Датасет
Humanoid Robot Navigation & Scheduling Dataset
Навигационная эффективность
Количество столкновений
Длина пути
Энергопотребление
Общая производительность (целевая переменная)
